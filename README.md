# Admin Panel

## Base Functionality

- Attach restaurant to user
- Add user
- Remove user
- Update password
- Change role
- Change seed votes
- Edit restaurant details
  - name
  - address
  - slug
  - status
  - geolocation

## Dashboard

  - # of restaurants in system
  - # claimed restaurants
  - map display of above
  - velocity/performance of sales
  - Avg # categories for menu
